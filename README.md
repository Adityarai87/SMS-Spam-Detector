# SMS-Spam-Detector
An ML-based NLP project that classifies SMS as Spam or Ham with 97% accuracy. It uses text preprocessing and Naive Bayes to automate the filtering of fraudulent and promotional messages.  Key Tech: Python, Scikit-learn, NLP (Vectorization).
SMS Spam Detection System 📩
Overview
This project is an NLP-powered Machine Learning solution designed to automatically classify SMS messages into Spam or Ham (Legitimate). It helps in filtering out fraudulent, promotional, and unwanted messages using intelligent text analysis.

Key Features
End-to-End NLP Pipeline: Includes data cleaning, tokenization, and text preprocessing.

Vectorization: Uses advanced techniques to convert raw text into numerical data for model training.

High Performance: Achieved an Accuracy of 97% and a Precision of 97.3% using the Naive Bayes algorithm.

Robust Security: Designed to protect users from phishing and scam-related messages.

Tech Stack
Language: Python

Libraries: Pandas, NumPy, Scikit-learn (ML), NLTK (NLP)

Deployment Ready: Includes a vectorizer.pkl and model file for seamless integration.

How it Works
Data Cleaning: Removes duplicates and handles missing values.

Preprocessing: Cleans text by removing special characters and stop words.

Model Training: Trains on a labeled dataset to recognize spam patterns.

Prediction: Classifies new incoming messages with high reliability.
